## 2026-02-26 (v2.0.0-beta)
**変更項目**: AirFuture PTPモデルの統合、金融シミュレーション実装、および投資判断3軸の明確化
**旧値**: 機器物販・権威ルート中心のピッチ
**新値**:
- 【PTP統合】「導入障壁ゼロ・高回転」を実現するプロフェッショナル体験プログラム（PTP）をPhase Aの主力エンジンとして実装。ROI 260%の経済性を明記。
- 【Sensory AI】MUI Roboticsのコア技術を「嗅覚/味覚データ化＋AI解析（Sensory AI）」として再定義し、技術的優位性を強化。
- 【財務計画実装】5ヶ年統合損益計画（P&L）をトグル形式で実装。2030年売上120億、営業益29億（EBIT）の推計値を反映。
- 【投資判断の3軸】「参入障壁 (Moat)」「スケーラビリティ」「ユニットエコノミクス」の3項目で投資合理性を総括。
- 【専門用語拡張】市場モメンタム、Moat、JV、PTP、PSE等の定義を追加し、非専門家への配慮。
- 【バグ修正】崩れていたグリッドレイアウトを完全修正し、レスポンシブ表示を安定化。
**目的**: プレゼンター不在でも事業の経済性と成長エンジン（PTP）が極めてリアルに伝わり、即時の投資判断を促しうる状態にするため。
