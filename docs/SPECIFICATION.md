# JV Project Pitch Document Specification

## 1. 概要 (Overview)
本ドキュメントは、WCA（World-CleanAir）とMUI Robotics（Thailand）によるジョイントベンチャー（JV）の資金調達用ピッチ資料（HTML形式）の設計仕様をまとめたものである。

## 2. コンセプト (Core Concept)
- **「We monetize devices. We scale with sensing. We dominate with data.」**
- 空気清浄機の販売会社ではなく、空気データ（リスクの可視化・改善・証明）のインフラ企業としてのポジショニング。

## 3. 主要セクション構成 (Page Structure)
1.  **Hero Section**: ビジョンと言語化されたミッションの提示。
2.  **Alliance Partners**: MUI（Sensory AI/センサー）とWCA（空気浄化/販路）の補完関係。
3.  **Trinity Cycle**: Measure（測定）→ Improve（改善）→ Certify（証明）の事業サイクル。
4.  **Products**: MUI Sensor（高精度センシング）とAirFuture（医療グレード浄化器）。
5.  **Strategy (Phase A-C)**: 
    - Phase A: 機器・PTP（体験導入）による初期キャッシュと権威獲得。
    - Phase B: SaaSモデルによるストック収益。
    - Phase C: データライセンス、API、ESG認証ビジネス。
6.  **Progress & Moat**: PoC実績、世界的権威（gIDE/ADA）との連携、データ蓄積による参入障壁。
7.  **Financials**: 5ヶ年損益計画（P&Lトグル実装）、調達目標（10億円）、バリュエーション（IPO 150-250億規模）。
8.  **Glossary**: 投資家が混乱しやすい用語（Moat, PTP, PSE等）の定義。

## 4. 収益エンジンの仕様 (Revenue Engine Logic)
### AirFuture PTP (Professional Trial Program)
- **モデル**: 月額2万円のレンタルから、購入時に利用料金を全額充当する「体験型クロージング」。
- **経済性**: 原価15万円、販売30万円。3年ROI 260%の資金効率。
- **データ戦略**: 試用期間中も環境データを収集し、AI教師データ（臭気ラベル）として資産化。

## 5. 財務計画 (Financial Projections)
### 2030年到達目標 (8月決算)
- **売上高**: 120.9 億円
- **営業利益 (EBIT)**: 29.2 億円
- **営業利益率**: 約24%
- **評価基準**: SaaS＋データ企業マルチプルによるDCF評価。

## 6. 技術仕様 (Technical Stack)
- **形式**: 単一HTMLファイル（`pitch.html`）
- **CSS**: バニラCSS (Modern Bento Grid Layout)、Glassmorphismテーマ。
- **レスポンシブ**: PC（1200px以上）、タブレット、スマホの全画面に最適化。
- **インタラクション**: セクションごとのホバーエフェクト（トグル動作検討中）。

- **v1.x**: 構築・デザイン最適化フェーズ
- **v2.0.0-beta (2026-02-26)**: PTPモデル、財務計画、Sensory AI定義を統合。
